# Augmenting education data
A tool used to create fake education data for normalizing education problem

## Requirements
1. unidecode
2. random
3. re
4. pandas
5. tqdm
6. numpy

## Version
Python 3.9.6

## Environment
- Anaconda, Windows 10
- Anaconda, Ubuntu 18.04

## Run
Open a terminal, run `python augment_education.py`
